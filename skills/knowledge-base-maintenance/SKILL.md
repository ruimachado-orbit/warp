# Knowledge Base Maintenance

When the same question appears repeatedly or the draft reply requires manual explanation, propose a KB update.

## Output

- Article/macro title
- Problem statement
- Step-by-step answer
- When to escalate
- Related tags/categories
- Source tickets/conversations
