Key Features

  • Instant ticket triage & routing across Zendesk, Intercom, Freshdesk, Salesforce, HubSpot, Jira, Front, Help Scout
  • Rule‑based classification with optional LLM‑draft replies
  • Knowledge‑base search & macro generation
  • Safe dry‑run mode – mutations require explicit --execute
  • Slack & Notion integration for escalation and documentation

python3 src/orchestrator.py "triage: production outage affecting all users"
