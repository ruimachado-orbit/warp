Multi-Platform

Works with 8+ major helpdesk systems

🚀 Capabilities

  • Ticket classification: category, severity, sentiment, priority
  • Smart routing to L1, L2, billing, privacy, integrations teams
  • Auto-draft customer replies and internal notes
  • Knowledge base search for macros and policies
  • SLA calculation and tracking
  • Multi-language support (EN, PT)
  • Safe execution with --execute flag requirement

🔌 Supported Integrations

Zendesk Intercom Freshdesk Salesforce HubSpot Jira Front Help Scout Slack Notion

🚀 Quick Install

# Via Homebrew (coming soon)
brew tap maio-labs/tap
brew install warp

# Or via install script (coming soon)
curl -fsSL https://raw.githubusercontent.com/maio-labs/warp/main/install.sh | bash

📦 Manual Install

git clone https://github.com/ruimachado-orbit/warp.git
cd warp
make setup
make dev

⚡ Quick Start

# Interactive chat mode
make chat

# Single query
make query Q="how do I reset my password?"

# Triage a ticket (rule-based, no LLM needed)
make triage T="customer says production is down"

# Check integration status
make status
